# Dialogue System in Unity


根据Brackeys的代码修改了一些。

DialogueManager
核心，处理如何显示等等，可以把它放在UIManager上

Dialogue
会话的数据结构，可以根据它进行修改

DialogueTrigger
会话触发器，包含了Dialogue结构

感谢Brackeys

Project files for a tutorial on creating a Dialogue System in Unity

Everything is made using Unity.
